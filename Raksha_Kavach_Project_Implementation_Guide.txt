
RAKSHA-KAVACH PROJECT IMPLEMENTATION GUIDE
Project No: 42

Project Title:
Android App Development using GenAI – Raksha-Kavach (Healthcare / Worker Safety)

==================================================
1. PROJECT OBJECTIVE
==================================================

Build an Android application for construction and factory workers
to improve workplace safety using:
- PPE (Safety Gear) Checklist
- Risk Level Detection
- Safety Awareness
- Incident Reporting

==================================================
2. MAIN FEATURES TO IMPLEMENT
==================================================

A. Splash Screen
- App Logo
- App Name

B. Login / Register
- Firebase Authentication
- Email & Password Login

C. Dashboard
Buttons:
- Start Safety Check
- Daily Quiz
- Report Incident
- Safety Score

D. Task Selection Module
Tasks Examples:
- Welding
- Electrical Work
- Height Work
- Digging Trench
- Chemical Handling

E. PPE Safety Checklist
Show required safety equipment:
- Helmet
- Gloves
- Boots
- Goggles
- Face Shield

Options:
- Wearing
- Not Wearing

F. Risk Meter
Show danger level:
- Green = Safe
- Yellow = Medium Risk
- Red = High Risk

G. Worker Avatar / Safety Illustration
- Use PNG images or Lottie animations
- Show worker with and without safety gear

H. Daily Safety Quiz
Simple MCQ Questions

Example:
Which gear is compulsory for welding?

I. Incident / Near Miss Report
Fields:
- Incident Title
- Description
- Date
- Optional Image Upload

J. Safety Score
- Increase score for safe days
- Reduce score for incidents
