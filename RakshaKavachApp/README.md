# Raksha-Kavach

Android app for construction and factory worker safety.

## Features
- Splash screen
- Local demo auth (no Firebase required)
- Task selection and PPE checklist
- Rule-based risk meter
- Daily safety quiz
- Incident reporting with optional image
- Safety score tracking

## Setup
1. Open the project in Android Studio.
2. Install Android SDK 34 and the Android Gradle plugin if prompted.
3. Sync Gradle and run on an emulator or device.

## Notes
- Demo mode is the default. No Firebase config is needed.
